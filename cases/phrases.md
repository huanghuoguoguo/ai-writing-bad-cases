# 常见短语与句式

## 1. 不是……而是……

- 类型：对比式金句模板
- 常见问题：太爱下定义，太像总结陈词，密度高时会显得像批量生成。
- 更像人的改写：直接说结论，或者用具体场景替代抽象对举。

示例：

- AI 味：真正重要的不是速度，而是你是否能长期坚持。
- 人写法：速度没那么重要，难的是你能不能一直做下去。

## 2. 稳稳接住你

- 类型：安抚型情绪表达
- 常见问题：拟人化和情绪服务感过强，容易显得悬浮、套路、短视频文案味重。
- 更像人的改写：写清楚具体能提供什么帮助，不用空泛承诺。

示例：

- AI 味：这套方法会稳稳接住你，让你从混乱里走出来。
- 人写法：这套方法至少能帮你先把信息整理清楚，再决定下一步怎么做。

## 3. 把 X 拉满

- 类型：口语化强情绪模板
- 常见问题：看起来很“会说网感话”，但重复使用后极其同质化。
- 更像人的改写：改成明确动作或程度描述。

示例：

- AI 味：把氛围感直接拉满。
- 人写法：现场的灯光和音乐一下子把气氛推起来了。

## 4. 你只管……，剩下的交给……

- 类型：广告式托管承诺
- 常见问题：夸张、失真、像投放文案，不像真实表达。
- 更像人的改写：交代边界和条件，不做全包式承诺。

示例：

- AI 味：你只管表达，剩下的交给模型。
- 人写法：你先把核心意思说清楚，剩下的润色可以交给模型处理。
